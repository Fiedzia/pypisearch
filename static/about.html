Idea

One day I was using pip search and I became frustrated with its speed and inflexibility, so I've set up local index. I am posting it online as it might be useful for other people as well.

Query syntax:

Full lucene syntax is exposed. Here is what you can do:
  • django AND git
  • django AND NOT git
  • (django OR flask) AND REST
  • django AND licence:bsd
  • django^10 OR flask^100

    • By default name, summary and description fields are searched, prioriterized in that order.
